 Ничего устанавливать не нужно, просто запустить exe файл в папке Resault и можно запускать.

Дополнительно
как откомпелировать файл python 3.X ?

1)нам нужна библиотека cx_Freeze, она позволяет скомпелировать файл с расширением py,
  подцепляет все использующиеся библиотеки, и таким образом данный файл можно запускать
  на компьютере, где НЕ УСТАНОВЛЕН интерпретатор python 3.X.
  устанавливается через pip (cmd/терминал => pip install "название библиотеки")

2)нужен сам файл, который надо скомпелировать (пусть будет называться test.py), 
  и нужен файл setup.py (я его подготовил, там всё описано)
  бросаем setup.py, test.py в одну папку
  открываем cmd/терминал, переходим в эту папку
  и пишем python setup.py build
  и в папке build всё будет готово!
